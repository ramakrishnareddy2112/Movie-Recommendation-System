# Movie Recommendation System

A Machine Learning based Movie Recommendation System built using Python, NLP, Scikit-learn, and Streamlit.

## Features
- Recommends similar movies
- Uses cosine similarity
- NLP-based text preprocessing
- Interactive Streamlit web app

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit

## Project Structure
- dataset/
- models/
- notebooks/
- app/

## Run the App

```bash
streamlit run app/app.py
```
